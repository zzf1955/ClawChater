# debug - Bug 修复工作流程

开始修复第一个待修复的 bug。

## 执行步骤

1. **读取 Bug**：从 `doc/BUGS.md` 的"待修复"区域读取第一个 bug
2. 将立即bug移动到"修复中"区域
3. **定位原因**：分析代码，定位 bug 产生的根本原因
4. **修复实现**：编写修复代码
5. **测试验证**：测试修复是否有效，确保不引入新问题
6. **更新日志**：测试通过后更新 `doc/CHANGELOG.md`
7. **完成归档**：将 bug 从"修复中"移到"已修复"

## 注意事项

- 确保修复不引入新问题
- 修复完成后必须更新 CHANGELOG.md
